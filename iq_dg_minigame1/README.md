Mini-code i wrote to solve last level of Defend the Castle! minigame in IQ Dungeon2. Can be modified to use in other levels (probably). 


# Battle Formation Solver

## Описание проекта
Программа для поиска оптимального размещения пяти боевых фигур (2 Polearm, 2 Sekire, 1 Chilivary) на игровом поле 6×5 с целью уничтожения всех врагов. Фигуры размещаются последовательно, каждая последующая фигура может быть установлена только после атаки предыдущей.

## Типы фигур

### 1. Polearm (Копейщик) ⚔️
Атакует 4 соседние клетки по горизонтали и вертикали:
```
    a
  a P a
    a
```
- Дальность атаки: 1 клетка
- Направления: вверх, вниз, влево, вправо

### 2. Sekire (Стрелок) 🏹
Атакует 4 клетки по диагонали:
```
a   a
  S
a   a
```
- Дальность атаки: 1 клетка
- Направления: по диагоналям

### 3. Chilivary (Рыцарь) 🛡️
Атакует все клетки по вертикали и горизонтали до первой стены:
```
      a
      a
      a
  a C a a a a
      a
      a
      a
```
- Атака блокируется стенами
- Бьет на неограниченное расстояние

## Правила размещения

1. **Последовательность**: Фигуры размещаются одна за другой
2. **Очередность атаки**: Первая фигура атакует, затем ставится вторая и т.д.
3. **Свободные клетки**: Фигура может стоять только на пустой клетке (не стена, не враг)
4. **Уничтожение врагов**: После атаки фигурой враги в зоне поражения исчезают
5. **Неуязвимые стены**: Стены нельзя разрушить атакой
6. **Цель**: Уничтожить всех 13 врагов, используя все 5 фигур

## Алгоритм решения

### Метод поиска с возвратом (Backtracking)

1. **Генерация перестановок**:
   - Все уникальные перестановки 5 фигур (2 Sekire, 2 Polearm, 1 Chilivary)
   - Всего 30 уникальных комбинаций

2. **Для каждой перестановки**:
   - Рекурсивно перебираем все возможные позиции для текущей фигуры
   - Проверяем валидность позиции (не стена, не враг)
   - Симулируем атаку и удаляем убитых врагов
   - Переходим к следующей фигуре
   - Если все враги уничтожены - решение найдено

3. **Оптимизации**:
   - Фигура должна убить хотя бы одного врага
   - Использование множеств для быстрых операций
   - Отсечение заведомо неперспективных веток


## Формат входных данных

### Параметры в коде
```python
# Размер поля
rows, cols = 6, 5

# Стены (координаты с 1)
walls = {(3,1), (4,2), (3,3), (4,4), (3,5)}

# Враги (координаты с 1)
enemies = {
    (4,1), (5,2), (4,3), (5,4), (4,5), (5,5), (5,1), (5,3),
    (6,1), (6,2), (6,3), (6,4), (6,5)
}
```
## Выходные данные

Программа выводит:
1. Статистику по перебору комбинаций
2. Найденные решения с детализацией:
   - Порядок размещения фигур
   - Координаты каждой фигуры
   - Кого убивает каждая фигура
   - Остаток врагов после каждого хода
3. Финальную карту с отображением:
   - `P` - Polearm
   - `S` - Sekire
   - `C` - Chilivary
   - `X` - Стены
   - `E` - Изначальные позиции врагов
   - `.` - Пустые клетки
